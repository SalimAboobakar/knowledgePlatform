## 🔧 كيفية تشغيل المشروع

### المتطلبات الأساسية
- Node.js (v16 أو أحدث)
- npm أو yarn
- حساب Firebase

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/SalimAboobakar/knowledgePlatform.git
cd knowledgePlatform
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **إعداد ملف البيئة**
```bash
cp env.example .env
# عدّل ملف .env بإضافة بيانات Firebase الخاصة بك
```

4. **تشغيل المشروع**
```bash
npm start
```
